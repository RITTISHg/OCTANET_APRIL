## 📖 About the Project

**Trenz** is a modern, responsive e-commerce landing page for an online fashion store. It showcases a curated collection of clothing for **Men**, **Women**, and **Kids**, featuring top brands and the latest trends — all in a clean, visually engaging layout.

This project was built as part of the **OctaNet April Internship Task** to demonstrate front-end web development skills using pure HTML, CSS, and JavaScript.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🖼️ **Image Slideshow** | Auto-rotating product slideshow with smooth fade animation |
| 🧭 **Navigation Bar** | Stylish top nav with category links and a search bar |
| 🏷️ **Top Banners** | Promotional offer banners for quick deal discovery |
| 👗 **Shop by Category** | Browse Men, Women & Kids sections at a glance |
| 🏆 **Best of Brands** | Dedicated section highlighting premium fashion brands |
| 📋 **Footer** | Informative footer with policy, help, and social links |
| 🎨 **Attractive UI** | Rich color palette with hover effects and smooth transitions |

## 🚀 Getting Started

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/RITTISHg/OCTANET_APRIL.git
   ```

2. **Open in browser**
   ```bash
   cd OCTANET_APRIL
   open Landing_page.html
   # or just double-click Landing_page.html in your file explorer
   ```

No build steps or dependencies required — it's pure front-end! 🎉
